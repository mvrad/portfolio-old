What I Do

I develop website designs with computer programming. I also work with technologies like databases and APIs in creating interactive web applications.


Front-End Development

I design web applications using responsive web design techniques and develop for the front end of websites in JavaScript libraries like React.


Back-End Development

I program for the back end of web applications in frameworks like Express and Django, all the while applying web application security best practices.


Development Tools

Other tools I use that enhance my workflow include Git and GitHub, Webpack–a JavaScript module bundler, Gulp–a front-end toolkit, and Adobe XD.

Who I Am

I'm Matt, a web developer based in Seattle. Computers have always fascinated me. My earliest memories of using a computer include playing with an old cartoon creation program for MS-DOS called Cartooners.
